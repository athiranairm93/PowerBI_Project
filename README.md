# PowerBI_Project
Strategic Sales &amp; Customer Insights Dashboard
The dashboard consist of five main pages, each focusing on a key aspect of the business.
1. Executive Trend Monitoring (Revenue, Profit & Growth)
   Managers can monitor long-term health and identify seasonal patterns using the trend dashboard.
   The dashboard tracks Year-over-Year (YoY) and Month-over-Month (MoM) growth to detect sudden market shift.
3. Segment Performance Analysis
   Primary revenue drivers by Segments
5. Regional & Store Performance (Underperforming Areas)
   Operations managers can identify stores that require attention based on profit margins rather than just volume.
7. Service Quality & Customer Retention
   This view correlates operational metrics with financial outcomes.
9. Impact Analysis
A dedicated “What-If” parameters pane allows managers to simulate the impact of strategy changes
